## MacExcelTool

一款将 excel 文件 (.xlsx) 与 txt 文件 (.txt) 互相转换的 Mac 应用。本应用主要使用了 Mac 程序与脚本之间的交互，通过调用脚本与监听事件完成相应的任务和进度提示。
